# UsedBay

## Project Purpose
A full-stack second-hand marketplace platform where users can buy and sell used items with role-based dashboards (Buyer, Seller, Admin), order management, and payment processing.

## Live URL
[https://usedbay.com](https://used-bay.vercel.app/)

## Key Features
- User authentication with email/password and Google OAuth
- Product listing and browsing with search and filtering
- Shopping cart and wishlist functionality
- Order management (buyers and sellers)
- Payment processing with Stripe
- Seller dashboard with analytics
- Admin panel for user and order management
- Order status tracking
- Sales analytics and charts
